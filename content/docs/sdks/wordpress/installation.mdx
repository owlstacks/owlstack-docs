# WordPress Installation

## Requirements

- WordPress 6.0+
- PHP 8.2+
- An OwlStack account and API key

## Install from WordPress.org

1. Go to **Plugins > Add New** in your WordPress admin
2. Search for **OwlStack**
3. Click **Install Now**, then **Activate**

## Install via Composer

```bash
composer require owlstack/wordpress
```

## Manual install

1. Download `owlstack.zip` from [owlstack.dev/download](https://owlstack.dev/download)
2. Go to **Plugins > Add New > Upload Plugin**
3. Upload the zip file and activate

## Configure

1. Go to **OwlStack > Settings** in the WordPress admin
2. Enter your **API key** (get one at [app.owlstack.dev](https://app.owlstack.dev))
3. Select which post types to show the publishing meta box on
4. Save settings

## Verify

Go to any post edit screen. You should see the **OwlStack** meta box in the sidebar.
