# Admin Panel

The OwlStack plugin adds an **OwlStack** top-level menu to the WordPress admin with the following pages:

## Dashboard

The main dashboard page shows:

- **Connection status** - whether your API key is valid
- **Connected platforms** - which social accounts are connected
- **Recent activity** - last 10 published posts with status

## Settings

Configure the plugin:

| Setting | Description |
|:--------|:------------|
| API Key | Your OwlStack API key |
| Post Types | Which post types show the OwlStack meta box |
| Auto-publish | Automatically publish when a post is published in WordPress |
| Default Platforms | Which platforms to publish to by default |

## Publishing Log

View a log of all publishing activity:

- Post title and link
- Platforms published to
- Status (success / failed)
- External URL (link to the social post)
- Timestamp

Filter by status, platform, or date range.
