# Meta Boxes

The OwlStack meta box appears on the post edit screen for enabled post types.

## Features

- **Platform toggles** - enable/disable individual platforms for this post
- **Custom message** - override the auto-generated social text
- **Preview** - see how the post will look on each platform
- **Tags/Hashtags** - add hashtags (defaults to post tags)
- **Media** - select which featured image or media to attach
- **Publish button** - manually trigger publishing

## Auto-publish

If auto-publish is enabled in settings, posts are automatically published to your selected platforms when:

- A post transitions from `draft` to `publish`
- A scheduled post goes live

You can disable auto-publish per post using the toggle in the meta box.

## Block Editor Support

The meta box is available as a **sidebar panel** in the block editor (Gutenberg). It appears under the OwlStack icon in the post settings sidebar.

## Classic Editor

For the classic editor, the meta box appears below the content editor as a standard WordPress meta box.
