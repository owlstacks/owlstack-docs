## How SDKs Work

All OwlStack SDKs are thin API clients that communicate with the OwlStack cloud service:

| Package | Language | Status |
|:--------|:---------|:-------|
| OwlStack Core (PHP) | PHP | v1.0 — Available |
| OwlStack Laravel | PHP / Laravel | v2.0 — Available |
| OwlStack WordPress | PHP / WordPress | v1.0 — Available |
| OwlStack Node.js | JavaScript / TypeScript | Coming Soon |
| OwlStack Python | Python | Coming Soon |
| OwlStack Go | Go | Coming Soon |
| OwlStack Ruby | Ruby | Coming Soon |
| OwlStack Rust | Rust | Coming Soon |

Don't see your language? You can use the [REST API](/api/overview) directly from any language.
