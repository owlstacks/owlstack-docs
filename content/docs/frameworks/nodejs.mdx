# Node.js SDK

<Callout type="warn" title="Coming Soon">
The OwlStack Node.js SDK is currently in development. Sign up for updates at [owlstack.dev](https://owlstack.dev) to be notified when it's available.
</Callout>

## What to Expect

The Node.js SDK will provide a native JavaScript/TypeScript client for the OwlStack API, designed for:

- **Node.js** backend applications
- **Express**, **Fastify**, **NestJS**, and other frameworks
- **Next.js**, **Nuxt**, and other full-stack frameworks
- **TypeScript-first** with full type definitions

## Preview

```typescript
import { OwlStack, Platform } from '@owlstack/node';

const client = new OwlStack({
  apiKey: process.env.OWLSTACK_API_KEY,
});

const result = await client.publish({
  content: 'Hello from Node.js!',
  platforms: [Platform.Twitter, Platform.LinkedIn],
  media: [{ type: 'image', path: './photo.jpg' }],
  hashtags: ['nodejs', 'typescript'],
});

console.log(result.successful); // Platform results
```

## In the Meantime

You can use the [OwlStack REST API](/api/overview) directly from any Node.js application using `fetch` or any HTTP client.

```typescript
const response = await fetch('https://api.owlstack.dev/v1/publish', {
  method: 'POST',
  headers: {
    'Authorization': `Bearer ${process.env.OWLSTACK_API_KEY}`,
    'Content-Type': 'application/json',
  },
  body: JSON.stringify({
    content: 'Hello from Node.js!',
    platforms: ['twitter', 'linkedin'],
  }),
});
```
