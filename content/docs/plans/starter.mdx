# Starter Plan

The Starter plan is free and includes everything you need to start publishing to social media.

## What's included

- **All 11 platforms** — Telegram, Twitter/X, Facebook, LinkedIn, Discord, Instagram, Pinterest, Reddit, Slack, Tumblr, WhatsApp
- **50 posts per month** — enough for personal projects and testing
- **Media attachments** — images, videos, GIFs
- **Automatic formatting** — platform-specific content formatting
- **Hashtag extraction** — automatic hashtag handling from tags
- **SDK access** — PHP SDK, Laravel package, WordPress plugin
- **Dashboard** — manage platforms and view activity

## Limitations

- No batch publishing (one post at a time)
- No scheduling
- No delivery logging
- No analytics
- No AI features

## Getting started

1. [Create an account](https://app.owlstack.dev)
2. Connect your platforms
3. [Install the SDK](/docs/getting-started/installation)
4. Start publishing!
