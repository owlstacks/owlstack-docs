# Creating Posts

OwlStack makes it easy to publish content to multiple social media platforms at once. Here's how.

## Writing a Post

1. Click **New Post** from the dashboard
2. Type your message in the editor
3. Your content will be automatically adapted for each platform's requirements (character limits, formatting, etc.)

## Adding Media

You can attach media to your posts:

- **Images**: JPG, PNG, GIF, WebP
- **Videos**: MP4, MOV
- **Documents**: PDF (supported on some platforms)

Simply drag and drop files into the editor, or click the **Add Media** button to select from your Media Library.

OwlStack automatically:
- Resizes images to fit each platform's requirements
- Converts formats when needed
- Handles platform-specific media limits (e.g., Instagram carousel limits)

## Selecting Platforms

Below the editor, you'll see all your connected platforms. Check the ones you want to publish to. You can also:

- **Select all** to publish everywhere
- **Preview** how your post will look on each platform before publishing

## Adding Hashtags

Type hashtags directly in your post, or use the **Hashtag** tool to:

- Add hashtags that get placed according to each platform's best practices
- Use AI-suggested hashtags (available on all plans)

## Publishing Options

| Option | What It Does |
|:-------|:-------------|
| **Publish Now** | Immediately publishes to all selected platforms |
| **Schedule** | Set a date and time for the post to go live |
| **Save as Draft** | Save your post without publishing |

## Post Status

After publishing, you can track the status of each post:

| Status | Meaning |
|:-------|:--------|
| **Published** | Successfully posted to the platform |
| **Scheduled** | Waiting to be published at the scheduled time |
| **Draft** | Saved but not published |
| **Failed** | Something went wrong — click for details |

## Tips for Great Posts

- **Keep it concise**: Twitter has a 280-character limit, but short posts perform well everywhere
- **Use media**: Posts with images or videos get significantly more engagement
- **Customize per platform**: Use the platform preview to adjust content for each network
- **Schedule strategically**: Use the calendar to spread posts throughout the week
