# Pro + AI Plan

The Pro + AI plan adds AI-powered content features on top of everything in Pro.

## Everything in Pro, plus

- **AI content generation** — generate platform-optimized posts from a topic or URL
- **AI post optimization** — improve existing content for engagement
- **AI auto-hashtags** — intelligent hashtag suggestions based on content analysis
- **AI content calendar** — AI-suggested posting schedule based on audience analytics
- **AI A/B testing** — generate content variants and track which performs best

## AI quotas

| Feature | Monthly Quota |
|:--------|:-------------|
| Content generations | 500 |
| Post optimizations | 1,000 |
| Auto-hashtag requests | 2,000 |
| Calendar suggestions | 100 |
| A/B test variants | 200 |
