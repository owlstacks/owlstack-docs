# Analytics

**Pro**

OwlStack tracks publishing performance and provides analytics across all your connected platforms.

## Dashboard

View analytics at **Project > Analytics** in the [OwlStack dashboard](https://app.owlstack.dev):

- **Publishing volume** — posts per day/week/month
- **Platform breakdown** — which platforms get the most posts
- **Success rate** — percentage of successful publishes
- **Engagement** — likes, shares, comments (where available)

## API access

```php
$analytics = $client->analytics([
    'from' => '2026-02-01',
    'to' => '2026-02-14',
    'group_by' => 'platform',
]);

foreach ($analytics as $platform => $stats) {
    echo "{$platform}:\n";
    echo "  Published: {$stats['published']}\n";
    echo "  Failed: {$stats['failed']}\n";
    echo "  Engagement: {$stats['engagement']}\n";
}
```

## Available metrics

| Metric | Description | Availability |
|:-------|:------------|:-------------|
| `published` | Number of successful publishes | All platforms |
| `failed` | Number of failed publishes | All platforms |
| `impressions` | Number of views | Twitter, LinkedIn, Facebook |
| `engagement` | Total interactions | Twitter, LinkedIn, Facebook |
| `clicks` | Link clicks | Twitter, LinkedIn |
| `shares` | Reposts/reshares | Twitter, LinkedIn, Facebook |
| `comments` | Replies/comments | Twitter, LinkedIn, Facebook |
