# Platform Overview

OwlStack supports **11 social media platforms** through a single API. Connect your accounts via OAuth in the [OwlStack dashboard](https://app.owlstack.dev), then publish from your code.

## How publishing works

```php
use OwlStack\Cloud\OwlStackClient;
use OwlStack\Post;
use OwlStack\Enums\Platform;

$client = new OwlStackClient(apiKey: env('OWLSTACK_API_KEY'));

$post = Post::create('Hello from OwlStack!')
    ->withHashtags(['php', 'opensource']);

// Publish to any connected platform
$results = $client->publish($post, [
    Platform::Telegram,
    Platform::Twitter,
    Platform::LinkedIn,
]);
```

OwlStack's cloud handles all platform API communication, formatting, media processing, and authentication for you.

## Platform-specific docs

Each platform has a dedicated page with features, options, and constraints:

- [Telegram](./telegram) - Channels, groups, media groups, inline keyboards
- [Twitter/X](./twitter) - Threads, polls, quote tweets
- [Facebook](./facebook) - Pages, scheduled publishing
- [LinkedIn](./linkedin) - Personal profiles and company pages
- [Discord](./discord) - Bot and webhook modes, rich embeds
- [Instagram](./instagram) - Carousels, Reels, Stories
- [Pinterest](./pinterest) - Board and section targeting
- [Reddit](./reddit) - Self and link posts, flair support
- [Slack](./slack) - Block Kit, bot and webhook modes
- [Tumblr](./tumblr) - NPF content blocks
- [WhatsApp](./whatsapp) - Template messages, documents
