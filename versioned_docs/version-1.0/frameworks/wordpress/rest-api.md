OwlStack registers AJAX endpoints for use from the WordPress admin panel and themes.

## Endpoints

| Action | Description |
|:-------|:------------|
| Test connection | Validates credentials for a specific platform |
| Manual publish | Publishes content to a specific platform on demand |
| Log management | Retrieves and manages delivery logs |

All endpoints are authenticated via WordPress nonces and require appropriate user capabilities.
