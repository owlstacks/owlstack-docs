# Platform Overview

OwlStack supports **11 social media platforms** through a unified API. Each platform has its own formatter, constraints, and features — but your code stays the same.

## How platforms work

Every platform implements `PlatformInterface`:

```php
interface PlatformInterface
{
    public function name(): string;
    public function publish(Post $post, array $options = []): PlatformResponseInterface;
    public function delete(string $externalId): bool;
    public function validateCredentials(): bool;
    public function constraints(): array;
}
```

## Platform setup pattern

All platforms follow the same setup pattern:

```php
use Owlstack\Core\Config\PlatformCredentials;
use Owlstack\Core\Http\HttpClient;

// 1. Create credentials
$credentials = new PlatformCredentials('platform-name', [
    'key' => 'value',
    // ... platform-specific credentials
]);

// 2. Create platform with HTTP client and formatter
$platform = new PlatformClass($credentials, new HttpClient(), new PlatformFormatter());

// 3. Register with the platform registry
$registry->register($platform);

// 4. Publish through the publisher
$result = $publisher->publish($post, 'platform-name', $options);
```
