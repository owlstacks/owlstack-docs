# Delivery Logging

import ProBadge from '@site/src/components/ProBadge';

<ProBadge />

Track every publishing attempt with full delivery records. Available on **Pro** plans and above.

## View delivery logs

```php
use OwlStack\Cloud\OwlStackClient;

$client = new OwlStackClient('your-api-key');

// Get recent delivery logs
$logs = $client->logs()->list([
    'limit' => 50,
    'platform' => 'twitter',
]);

foreach ($logs as $log) {
    echo "{$log->platform} | {$log->status} | {$log->createdAt}\n";
}
```

## Log details

Each delivery log includes:

| Field | Description |
|:------|:------------|
| `id` | Unique delivery ID |
| `platform` | Platform name |
| `status` | `published`, `failed`, `pending`, `scheduled` |
| `externalId` | Platform's post ID |
| `externalUrl` | URL to the published post |
| `error` | Error message (if failed) |
| `errorCode` | Error classification |
| `attempts` | Number of delivery attempts |
| `createdAt` | When the publish was initiated |
| `publishedAt` | When delivery was confirmed |

## Filtering and search

```php
$logs = $client->logs()->list([
    'platform' => 'twitter',
    'status' => 'failed',
    'from' => '2026-01-01',
    'to' => '2026-01-31',
]);
```

## Dashboard

Delivery logs are also available in the [OwlStack dashboard](https://app.owlstack.dev) with filtering, search, and export capabilities.

## Retry failed deliveries

```php
// Retry a specific failed delivery
$client->logs()->retry($logId);
```
