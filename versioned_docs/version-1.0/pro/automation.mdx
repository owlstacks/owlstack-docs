# Automation

import ProBadge from '@site/src/components/ProBadge';

<ProBadge />

Set up rules that automatically publish content when triggered by events in your application.

## Features

- **Rule engine** — Define conditions and actions
- **Event triggers** — React to application events
- **Platform targeting** — Specify which platforms to publish to per rule
- **Conditional logic** — Publish based on content type, tags, categories, etc.

## Laravel

```bash
# List automation rules
php artisan owlstack:automation:list
```
