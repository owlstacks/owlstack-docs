# Templates

Create reusable templates with placeholder resolution for consistent content across platforms.

## Features

- **Placeholder resolution** — `{title}`, `{url}`, `{tags}`, custom placeholders
- **Per-platform templates** — Different templates for different platforms
- **Template inheritance** — Base templates with overrides
